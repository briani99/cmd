-----------------
STARTING AURELIA
-----------------

npm install -g aurelia-cli
au new  (New Project Wizard)
npm install
au run

au run --watch  (this runs your project and watchs for changes)
au run --env stage --watch  (use --env for dev,stage,prod)

au build --env stage

au test
au test -watch

au v
au help
